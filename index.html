PCBs

Isoduino MEGA


Isduino MEGA is an Arduino MEGA clone with added functionality, notably the ability to isolated the USB communication between the computer and the microcontroller, hence the name.

The Isoduino MEGA is compatible with the Arduino eco-system and allow the computer to be shield from the noisy component such as motors or relays.

Features:

  • Isolated UART to protect computer from motor noise.
  • Switching regulator to allow bigger range of voltage input.
  • Extra IO pins where broken out.
  • User cofigurable jumper to bypass isolation.
  • Automatic and dynamic voltage sourcce selection in Bypass Mode.
  • Voltage divider to allow input voltage sensing.
  • Extra user LED.

Ground Station Gimbal


The ground station gimbal has 1-degree of freedom to allow the atenna to be pointed toward a fixed-wing aircraft.

PCB is a Raspberry Pi HAT and allow the Raspbery Pi to communicate and control various devices on-board via I2C bus.

Features:

  • 9-axes imu (MPU9250) for orientation and absolute heading.
  • GPS (FGPMMOPA6C) for absolute location.
  • PWM controller (PCA9633) to generate motor control signals.
  • 5V buck convertor (TPS54302) to back power RPi with 3A max output.
  • Three-phase motor driver (L6234) for driving the motor.
  • Real time clock (DS3231M) for time keeping.
  • Current sensor (INA219) to monitor total current consumption and input voltage.

Battery Indicator


A simple battery indicator to monitor the battery level.

With a user press of botton, the battery remained is brightly displayed.

Features:

  • Voltage regulator allowing sensing up 35V.
  • ATtiny45 microcontroller with 98% flash usage.
  • Six WS2812B tri-color addressable LEDs.
  • Exponential filter to smooth out voltage read.
